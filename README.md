📊 Product Analytics & A/B Testing Project
📌 Overview

This project analyzes user behavior in an e-commerce platform to understand conversion patterns and evaluate product changes using A/B testing.

🎯 Objectives
Analyze user funnel (view → cart → purchase)
Identify drop-offs in conversion
Conduct A/B testing to evaluate performance differences
🛠️ Tools Used
Python (Pandas, NumPy)
Matplotlib
Statistical Testing (Z-test)
📊 Key Analysis
🔹 Funnel Analysis
Significant drop-off observed between product views and purchases
🔹 Conversion Rates
Calculated stage-wise conversion rates
🔹 A/B Testing
Conducted proportion Z-test
P-value: 0.83 → No significant difference
💡 Key Insights
Most users drop before completing purchase
No measurable impact between tested variants
📌 Business Recommendations
Improve checkout experience
Test stronger product/design changes
Continue experimentation
