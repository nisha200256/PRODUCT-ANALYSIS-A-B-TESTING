# 📊 Product Analytics & A/B Testing | Data Analytics Project

## 📌 Project Overview
This project analyzes user behavior and product performance using data analytics and statistical testing techniques. The focus is on identifying conversion bottlenecks and evaluating product changes using A/B testing to support data-driven decision-making.

---

## 🎯 Business Problem
Understanding user behavior across the product funnel is critical for improving conversion rates and optimizing product performance. This project aims to:
- Identify drop-off points in the user journey
- Measure conversion rates across funnel stages
- Evaluate product changes using A/B testing
- Provide actionable recommendations to improve user engagement

---

## 📈 Key Features
- 📊 Funnel Analysis (View → Cart → Purchase)
- 📉 Conversion Rate Calculation at each stage
- 🧪 A/B Testing using statistical hypothesis testing (Z-test)
- 📊 Data visualization for trend and pattern identification
- 🔍 Insight-driven analysis for product optimization

---

## 🧠 Key Insights
- Significant drop-off observed between cart and purchase stages
- Conversion rates vary across user segments, indicating targeting opportunities
- A/B test results show statistically significant improvement in conversion for optimized variant
- Data suggests improving checkout UX can increase overall revenue

---

## 🛠 Tools & Technologies
- Python (Pandas, NumPy)
- Statistical Testing (Z-test, p-value analysis)
- Data Visualization (Matplotlib, Seaborn)
- Jupyter Notebook

---

## 📊 Methodology

### 1. Data Preparation
- Cleaned and structured raw user interaction data
- Handled missing values and ensured data consistency

### 2. Funnel Analysis
- Calculated user progression through each stage
- Identified key drop-off points

### 3. A/B Testing
- Defined null and alternative hypotheses
- Conducted Z-test to compare control vs variant
- Evaluated statistical significance using p-values

### 4. Insights & Recommendations
- Interpreted results to provide actionable business insights
- Suggested improvements for product and user experience
